Provides the classes required to support concurrent requests to resources. This package contains various classes including a Singleton (ChannelManager) that governs all incoming requests and returns the resource when it becomes available.

Package Specification

Provides the classes required to perform concurrent resource requests to a number of channels. Each Channel object is unique based upon the request properties it is responsible for providing a resource to. The ChannelHost object is responsible for retrieving the resource and applying it to the channel.

When a channel has returned a resource to its last requesting ChannelSubscriber an attempt to perform a cleanup of the channel is initiated via a ChannelRecycler object.
If no further requests to that channel are received within a specified time frame, the Channel will be recycled. If a request synonymous to that channel is received within the time frame the recycling process is cancelled.
