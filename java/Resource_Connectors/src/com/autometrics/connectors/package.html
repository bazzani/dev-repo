Provides the classes required to create and use data connectors. This package contains various classes and interfaces, including a Factory class that will instantiate and return an implementation of the IDataConnector interface which all data connectors must implement.

Package Specification

Provides the classes required to construct data connectors. Data connectors provide an interface to connect to a variety of data sources within the Autometrics platform.
